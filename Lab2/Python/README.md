Створення середовища: python -m venv venv 
Активація середовища: Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser 
venv\Scripts\Activate.ps1

Запуск файлу requests_program.py:
1. Встановити бібліотеку requests за допомгою команди pip install requests.
2. За потреби виконати такі команди (якщо програма не виконується через відсутність бібліотеки): pip install --upgrade pip та pip install --upgrade requests.
3. Виконати код.

Встановлення залежносте:
1. Створити віртуальне середовище та активувати його.
2. Виконати команду: pip install -r requirements.txt.
